\# 🏠 California Housing Market Analysis



An end-to-end Data Analytics project analyzing California housing prices using Python, Scikit-Learn, and Power BI.



\---



\## 📌 Project Overview



This project analyzes \*\*20,640 California housing records\*\* to identify the key factors influencing house prices.



The workflow includes:



\- Data Cleaning

\- Exploratory Data Analysis (EDA)

\- Feature Engineering

\- Correlation Analysis

\- Linear Regression

\- Geographic Analysis

\- Interactive Power BI Dashboard

\- Business Recommendations



\---



## 🛠 Technologies Used

### Programming & Analysis

- Python
- Pandas
- NumPy
- Scikit-Learn

### Visualization

- Matplotlib
- Seaborn
- Power BI

## 📂 Dataset

- **Source:** California Housing Dataset (Scikit-Learn)
- **Records:** 20,640
- **Target Variable:** House Value
- **Features:** 8 numerical features including Median Income, House Age, Population, Latitude, and Longitude.

\## 📈 Key Insights



\- Median Income is the strongest predictor of property values.

\- Coastal regions have significantly higher-priced properties.

\- Property values increase consistently across income groups.

\- House Age has relatively little influence compared to income.

\- Housing prices are capped near \*\*$500,000\*\*, indicating a dataset limitation.



\---



\## 💼 Business Recommendations



\- Focus premium housing developments in affluent coastal regions.

\- Use income segmentation while designing pricing strategies.

\- Prioritize Median Income in predictive pricing models.

\- Invest in high-income coastal markets.

\- Incorporate geographic features into future pricing models.

## ▶️ How to Run

1. Clone this repository.
2. Install the required libraries:

```bash
pip install -r requirements.txt
```

3. Open `California_Housing_Analysis.ipynb` in Jupyter Notebook or JupyterLab.
4. Run the notebook sequentially to reproduce the complete analysis and machine learning workflow.
5. Open `California_Housing_Dashboard.pbix` in Power BI Desktop to explore the interactive dashboard.
